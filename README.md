# Voltura - Setup Guide

## Requirements

Node.js >= 20.19.0 or >= 22.12.0

npm >= 9.x

## Run

To start both the JSON server and the Vite development server at the same time, run:

```
npm run start:all
```

JSON server will run on http://localhost:3000

Vite dev server will run on http://localhost:5173
